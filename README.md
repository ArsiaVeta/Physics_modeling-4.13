# Моделирование полёта баллистической ракеты

##  Цель работы

* Построить математическую и дискретную модель движения баллистической ракеты
* Реализовать численный алгоритм решения уравнений движения
* Провести вычислительный эксперимент
* Проанализировать траекторию и скоростные характеристики
* Выяснить условия достижения первой космической скорости

---

##  Математическая модель

Используется следующая система дифференциальных уравнений:

```
dx/dt = v·cosθ
dy/dt = v·sinθ
dv/dt = (αu − Fсопр)/m − g·(R²/(R+y)²)·sinθ
dθ/dt = − g·(R²/(R+y)²)·cosθ / v
```

Сила сопротивления:

```
Fсопр = k·v²,   k = 0.5·c·S·ρ
```

Плотность атмосферы:

```
ρ = ρ₀·10^(−βy)
```

Закон изменения массы:

```
m(t) = max(m₀ − αt, mкон)
```

---

##  Исходные данные

* Начальная масса: `m₀ = 6·10⁵ кг`
* Конечная масса: `mкон = 10⁴ кг`
* Расход топлива: `α = 1000 кг/с`
* Скорость истечения: `u = 2500 м/с`
* Радиус Земли: `R = 6.37·10⁶ м`
* Ускорение свободного падения: `g = 9.81 м/с²`

Начальные условия:

```
x(0)=0, y(0)=0, v(0)=1 м/с, θ(0)=π/2
```

---

##  Численный метод

Для решения системы уравнений применена **явная разностная схема Эйлера**:

```
xₙ₊₁ = xₙ + vₙ cosθₙ Δt
yₙ₊₁ = yₙ + vₙ sinθₙ Δt
vₙ₊₁ = vₙ + dvₙ Δt
θₙ₊₁ = θₙ + dθₙ Δt
```

Шаг интегрирования: `Δt = 0.05 с`

##  Первая космическая скорость

Первая космическая скорость определяется формулой:

```
v₁ = √(gR) ≈ 7.9 км/с
```

В ходе моделирования максимальная скорость ракеты составила:

```
v_max ≈ 0.72 м/с
```

###  Вывод

Первая космическая скорость **не достигается** при заданных параметрах, так как:

* тяга двигателя недостаточна;
* начальная масса ракеты слишком велика;
* значительные потери скорости из-за силы тяжести и сопротивления атмосферы;
* отсутствует оптимальный этап формирования горизонтальной скорости.
