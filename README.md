# Physics_modeling-4.13

## Тема работы

Моделирование полета баллистической ракеты с учетом переменной массы, силы сопротивления атмосферы и изменения гравитационного поля Земли.

---

## Цель работы

Целью работы является построение математической и дискретной модели полета баллистической ракеты, разработка алгоритма численного решения системы дифференциальных уравнений движения и проведение вычислительного эксперимента для анализа траектории и скоростных характеристик ракеты, а также определение условий достижения первой космической скорости.

---

## Постановка задачи

Требуется:

1. Построить дискретную модель уравнений движения баллистической ракеты.
2. Реализовать алгоритм численного решения на языке программирования Python.
3. Провести моделирование полета при заданных параметрах ракеты.
4. Построить графики зависимостей:

   * ускорения от времени a(t);
   * скорости от времени v(t);
   * массы от времени m(t);
   * траектории полета y(x);
   * скорости от высоты v(y).
5. Определить, достигается ли первая космическая скорость.

---

## Математическая модель

Движение ракеты описывается следующей системой дифференциальных уравнений:

[
\begin{aligned}
\frac{dx}{dt} &= v \cos \theta, \
\frac{dy}{dt} &= v \sin \theta, \
\frac{dv}{dt} &= \frac{\alpha u - F_{сопр}}{m(t)} - g \frac{R^2}{(R+y)^2} \sin \theta, \
\frac{d\theta}{dt} &= - g \frac{R^2}{(R+y)^2} \frac{\cos \theta}{v}.
\end{aligned}
]

Сила сопротивления воздуха:
[
F_{сопр} = k v^2, \quad k = \frac{1}{2} c S \rho.
]

Плотность атмосферы изменяется по закону:
[
\rho = \rho_0 10^{-\beta y}.
]

Закон изменения массы ракеты:
[
m(t) = \max(m_0 - \alpha t,, m_{конс}).
]

---

## Исходные данные

* Начальная масса ракеты: ( m_0 = 6 \cdot 10^5 ) кг
* Конечная масса ракеты: ( m_{конс} = 10^4 ) кг
* Расход топлива: ( \alpha = 1000 ) кг/с
* Скорость истечения газов: ( u = 2500 ) м/с
* Радиус Земли: ( R = 6.37 \cdot 10^6 ) м
* Ускорение свободного падения: ( g = 9.81 ) м/с²
* Коэффициент сопротивления: ( c = 0.5 )
* Площадь поперечного сечения: ( S = 10 ) м²

Начальные условия:
[
x(0)=0, \quad y(0)=0, \quad v(0)=1, \text{м/с}, \quad \theta(0)=\frac{\pi}{2}.
]

---

## Численный метод

Для численного решения системы дифференциальных уравнений применена явная разностная схема Эйлера. Шаг интегрирования по времени выбран равным ( \Delta t = 0.05 ) с. Интегрирование проводится до достижения максимального времени моделирования или до момента падения ракеты.

---

## Результаты моделирования

В ходе вычислительного эксперимента были получены следующие графики:

* зависимость ускорения ракеты от времени a(t);
* зависимость скорости ракеты от времени v(t);
* изменение массы ракеты во времени m(t);
* траектория полета ракеты y(x);
* зависимость скорости от высоты v(y).

Анализ графиков показывает, что в начальный момент полета скорость и ускорение быстро возрастают за счет работы двигателя. По мере выгорания топлива масса ракеты уменьшается, а влияние силы сопротивления и силы тяжести возрастает.

---

## Первая космическая скорость

Первая космическая скорость определяется формулой:
[
v_1 = \sqrt{gR} \approx 7900 \text{ м/с}.
]

По результатам моделирования максимальная скорость ракеты сравнивается с первой космической скоростью. Если ( v_{max} \ge v_1 ), то условия достижения первой космической скорости выполняются.
